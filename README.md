# 数据结构与算法课程代码合集

这是我数据结构与算法课程的代码仓库，包含了课程中完成的实验、作业和课程设计。

## 项目结构

### 📚 Experiment - 常规实验
基础数据结构的实现和练习，共13个实验：
- 递归函数、排序算法
- 数组、链表、栈、队列
- 哈希表、二叉树、堆
- 二叉搜索树、图、最小生成树

### 🚀 Curriculum design - 课程设计
进阶项目，包含5个大作业：
1. **Exp1_Skiplist** - 跳表实现
2. **Exp2_External Sorting** - 外排序
3. **Exp3_File Directory System** - 文件目录树
4. **Exp4_The K Shortest Path** - K短路算法
5. **Exp5_Bus Route Optimization Query** - 公交线路优化查询（抽签题目）

### 📝 Homework - 平时作业
按日期整理的课程作业

## 说明

代码主要使用 C/C++ 编写，部分项目（如公交线路查询）还包含 Python 后端和前端界面。

欢迎参考学习，但请不要直接抄袭作业哦~
